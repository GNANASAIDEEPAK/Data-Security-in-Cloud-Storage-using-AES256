# 🔐 Data Security in Cloud Storage using AES256

## 📌 Project Description
This project provides a secure cloud storage system using AES-256 encryption. It ensures that user data is protected before storing it in the cloud.

## 🚀 Features
- 🔒 AES-256 encryption for secure data storage
- 👤 User authentication system
- 📁 File upload and download
- 🔑 Secure key management
- 📊 Admin and user dashboards

## 🛠️ Technologies Used
- Python
- Django
- HTML, CSS, JavaScript
- SQLite / Database

## 📂 Project Structure
- adminapp → Admin functionalities
- userapp → User functionalities
- assets → Static files
- templates → HTML files
- DATABASE → Database files

## ▶️ How to Run
1. Clone the repository
2. Install requirements:
   pip install -r Requirements.txt
3. Run server:
   python manage.py runserver
